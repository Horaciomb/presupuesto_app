# 💰 Gestor de Presupuesto Personal

Sistema de gestión de presupuesto personal desarrollado en Python con principios SOLID y POO. Permite registrar gastos, controlar deudas, generar reportes y visualizar gráficos.

## 📋 Características

- ✅ Registro de gastos fijos y variables
- ✅ Control de deudas en cuotas
- ✅ Seguimiento de ahorro mensual
- ✅ Fondo para imprevistos
- ✅ Reportes detallados en consola
- ✅ Gráficos de distribución y evolución
- ✅ Persistencia de datos en JSON
